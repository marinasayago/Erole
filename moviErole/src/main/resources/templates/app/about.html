About MoviErole

Some info about this website

Hi, we are Erole. Thanks for checking out our page!


We are a team formed by Software Engineering students at the University of Málaga, and this website is the result of a three-months project for the subject 'Introduction to Software Engineering'. The main idea was to develop a web application that focuses on entertainment.
What we brought up is a website where you can look up any movies or series, so you could consult basic information about them, giving you the chance to interact with other people within that specific content through a comment section, so you could express yourself and write a review or have discussions with other users about some topics.
That's how MoviErole was born!

                                                                                                     

Usage

Before you can use this service, you must have an account. In case you don't, you'll have to sign up to register your user (Get Started).
Once you've successfully signed up (a message shows on the main page), you'll be able to log in and use the services our website provides.


Technical aspects

This website was developed using the Spring Framework.
In order to make queries, this application is connected to an IMDb API, which is a web service for movies and series information. This API provides the results as JSON files.
We also used some tools from Bootstrap to improve the visual aspect of this website
As far as programming languages go, MoviErole was implemented making use of Java, HTML and Javascript.
